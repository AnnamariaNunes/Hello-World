O que é o Netbeans?

O NetBeans é um ambiente de desenvolvimento integrado (IDE) Java desenvolvido pela empresa Sun Microsystems.


Porque utilizar o Netbeans ?

O NetBeans IDE oferece aos desenvolvedores todas as ferramentas necessárias para criar aplicativos profissionais de desktop,
 empresariais, Web e móveis multiplataformas, de forma descomplicada e altamente prático.
